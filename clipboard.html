System Clipboard

Authors: Amrit Aneja and Wayne Rasband
History: 2001/12/13: First version
2002/04/11: Works with Java 1.4.0
2004/02/04: No longer opens multiple "Clipboard" windows
2004/02/06: Works with Java 1.5
2004/06/25: "Paste" works on Mac OS X
2004/09/28: "Clipboard" window can be opened more than once
2006/02/02: Does not block ImageJ keyboard shortcuts
Requires:Java 1.4 or later, Java 1.4.2 or later on Mac OS X
Limitations:Pasting on Mac OS X may cause ImageJ to crash
Source: System_Clipboard.java
Installation: Download System_Clipboard.class to the plugins folder and restart ImageJ.
Description: This plugin supports Copy/Paste of images to other applications. Click on the "Copy" button to copy the active image or selection to the system clipboard. Click on "Paste" to open the image on the system clipboard in an ImageJ window.
