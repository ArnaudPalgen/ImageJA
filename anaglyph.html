Two Shot Anaglyph

Author: Ronald Petie (Ronald.Petie at wur.nl)
History: 2005/06/03: First version
Source: Two_Shot_Anaglyph.java
Installation: Download Two_Shot_Anaglyph.class to the plugins folder and restart ImageJ.
Description: This plugin uses two photographs, made at different angles, to make a stereo image. The stereo effect is visible when using red-green glasses. This plugin is based on information about stereo photography found on www.photostuff.co.uk/stereo.htm.
