Gaussian Blur 3D

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/03/25: First release
Requires: ImageJ 1.46j or later
Source: Gaussian_Blur_3D.java
Installation: This plugin is built into ImageJ 1.46j or later as the Process>Filters>Gaussian Blur 3D command.
Description: This plugin does 3D gaussian blurring.
