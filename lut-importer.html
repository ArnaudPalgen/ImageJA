LUT Importer

Author: Wayne Rasband (wsr at nih.gov)
Date: 2001/11/30
Source: LUT_Importer.java
Installation: Download LUT_Importer.class to the plugins folder and restart ImageJ.
Description: This plugin imports a 256 entry LUT from a 3 or 4 column text file, such as those created by the LUT_Lister plugin. The LUT is added to the current image or, if no image is open, a new 256x32 image is created to display the LUT.

Update: The File/Import/LUT command in ImageJ 1.31b or later also opens LUTs that are in text format.
