Common Commands Menu Tool

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/03/03: First release
Requires ImageJ 1.46h or later
Source: CC_Menu_Tool.txt
Installation: Doanload CC_Menu_Tool.txt to the ImageJ/plugins/Tools folder, run the Help>Refresh Menus command, then select CC Menu Tool from the toolbar's ">>" drop down menu.
Description: This is a macro tool that adds a customizable drop down menu to the ImageJ toolbar that contains commonly used ImageJ menu commands. All the commands lack keyboard shortcuts and many are normally buried in sub-menus. Commands can be added to, or deleleted from, the drop down menu by editing the "cmds" array in the macro.
