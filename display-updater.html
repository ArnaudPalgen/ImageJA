Display Updater

Author: Wayne Rasband (wayne@codon.nih.gov)
Date:2000/6/14
Source: Display_Updater.java
Installation: Download Display_Updater.class to the plugins folder and restart ImageJ.
Description: This plugin displays a dialog box that allows the user to change the active image's minimum and maximum displayed values. For 8-bit images, these values are used to create a new LUT. For 16-bit images, they define how 16-bit pixel values are mapped to 8-bit display values. RGB images are not supported.

The Java source code provides an example of how to use ImageJ's GenericDialog class to create a dialog box with two numeric fields and one checkbox.
