Enclose

Author: A. Karperien, Charles Sturt University, Australia
History: 2002/08/26: First version
Requires: Java 2
Source: enclose_.java
Installation: Download enclose_.class to the plugins folder and restart ImageJ.
Description: This plugin encloses an image in a convex hull then calculates circularity and roundness. It uses the PolygonROI function by Thomas Roy, University of Alberta, Canada. It counts pixels at foreground value so is suitable for binary images, but will automatically threshold an image for counting. It analyzes multiple images one by one by opening them and using ROIs. Output to ImageJ's results window.
