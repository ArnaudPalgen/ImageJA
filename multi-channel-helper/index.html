Multi-Channel Image and Stack Helper

Author: Gilles Carpentier (image.bio.methods at free.fr)
History: 2012/04/05: First release
Source: Multi-Channel Image and Stack Helper
Installation: Download Multi-Channel Image and Stack Helper into the ImageJ/macros/toolsets folder and it will appear in the ImageJ toolbar's >> menu.
Description: This is a macro toolset for working with composite color images and hyperstacks. More information is available on the Multi-Channel Image and Stack Helper website.
