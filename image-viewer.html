Image Viewer

Author: Kurt DE Vos (kd269@columbia.edu)
History: 2000/11/15: First version
Source: Included in the ImageViewer.zip archive
Installation: Expand the ImageViewer.zip archive in the plugins folder and restart ImageJ. The plugins will be listed in the "UCSD Plugins" submenu of the Plugins menu.
Description: When the Image_Viewer is called, a system file tree is displayed. When an imagefile is double-clicked, it is opened after closing all other ImageWindows first. All the image formats that IJ supports can be opened. In addition it supports FluoView tiff images by making use of the USCD fluoview plugins. A checkbox allows to toggle between multichannel or single channel display.
