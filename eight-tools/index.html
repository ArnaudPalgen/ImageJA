Eight Tools

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/01/01: Initial release
2012/10/29: Fixed tool restoration and logging bugs
Requires ImageJ 1.47e or later
Source: Eight_Tools.java
Installation: Drag and drop Eight_Tools.java onto the "ImageJ" window, save it in plugins/Tools, then compile and run it using Plugins>Compile and Run.
Description: This plugin demonstrates how to add multiple custom tools to the ImageJ toolbar.
