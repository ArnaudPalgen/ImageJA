FFTJ and DeconvolutionJ

Author: Nick Linnenbrügger (nilin@web.de)
History: 2001/11/28: First version
2002/02/21: New features; uses less memory; runs on Macs
Source: Included in the fftj.zip ZIP file.
Installation: Download fftj.zip, extract the FFTJ folder into the plugins folder, then restart ImageJ.
Description: FFTJ is a generalized ImageJ plugin for the calculation of Fourier Transforms. It handles forward and inverse transformations of arbitrary-sized three-dimensional (3D) volumes as well as single two-dimensional (2D) images. Details.

DeconvolutionJ effectively implements deconvolution based on a Regularized Wiener Filter, as described in Gonzales&Woods: "Digital Image Processing", Chapter 5. It can handle arbitrary-sized three-dimensional (3D) volumes as well as single two-dimensional (2D) images. Details.
