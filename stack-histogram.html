Stack Histogram

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/6/14: First version
2002/04/01: Works with ImageJ 1.27s or later
2002/04/04: Works with density calibrated images
2004/02/12: Updated to work with ImageJ 1.31
2004/03/02: Updated to work with byte masks used by ImageJ 1.32
2008/05/14: Fixed binning bug
Source: Stack_Histogram.java
Installation: This plugin is built into ImageJ as the Analyze>Histogram command.
Description: This plugin calculates and displays the histogram of an entire stack.
