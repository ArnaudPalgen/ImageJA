MetaData (Line Scan Analysis)

Author: Sean Parsons (sparsons_llabh@yahoo.co.uk)
History: 2010/08/05: first version
Source: sourcex.tar.gz.
Requires: ImageJ 1.43 or above.
Installation: Download LineScan_Analysis.jar, flanagan.jar, jtransforms.jar and marfmath.jar to the plugins folder.
Description: The MetaData plugin saves and opens tiff images ("metapictures") and text files ("metatables") with metadata fields. Also there are several algorithms for analysis of confocal line scans of intracellular calcium - filtering, normalization and spark detection. See MetaData.pdf for a brief guide

MetaData interface.

Output of Confinement Tree algorithm.
