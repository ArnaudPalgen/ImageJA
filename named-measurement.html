Named Measurement

Author: Audrey Karperien (akarpe01 at postoffice.csu.edu.au)
History: 2003/09/03: First version
2003/12/11: Added enhanced version
Source: Named_Measurement.java
Installation: Download Named_Measurement.class to the plugins folder and restart ImageJ. Use the Plugins/Shortcuts/Create Shortcut command to create a keyboard shortcut.
Description: This plugin is an extended version of the Analyze/Measure command that allows naming of individual measurements.

An extended version of this plugin (Named_Measurement2) is also available. It allows naming of individual measurements, it labels ROIs on the image and it saves the ROIs for repeated processing. It is also customizable for processing in the DoSomethingOnThisRoi function (which currently has a dummy function. To use it, download Named_Measurement2.java to the plugins folder and compile it using Plugins/Compile and Run.
