Time Stamper

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/06/22: First version
2001/06/28: Picks up location from ROI
2001/11/04: Does not display time if interval is 0
2003/06/30: Font size is based on selection height
2005/05/10: Starting and ending frame can be specified
Source: Time_Stamper.java
Installation: This plugin is built into ImageJ 1.43t and later as the Image>Stacks>Label command.
Description: This plugins adds time stamps to a stack. The times are drawn in the current foreground color. Use the color picker (Image/Color/Color Picker) to set the foreground color. A dialog box allows the user to specify the starting time, time between frames, location, font size, decimal places and unit of time. Create a rectangular selection and the X and Y locations in the dialog box will be based on that selection. Set time between frames to zero to display nothing but the text in the Suffix field.
