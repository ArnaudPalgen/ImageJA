Measure And Label

Author: Wayne Rasband (wsr at nih.gov)
Date:2003/02/06
Source: Measure_And_Label.java
Installation: Download Measure_And_Label.class to the plugins folder and restart ImageJ.
Description: This plugin measures the current selection, outlines it in white, and labels it with the measurement number. Use Plugins/Shortcuts/Create Shortcut to create a keyboard shortcut that runs this plugin.
