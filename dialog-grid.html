Dialog Grid Demo

Author: Wayne Rasband (wsr at nih.gov)
History: 2003/11/13: First version
Requires:ImageJ 1.31l or later
Source: Dialog_Grid_Demo.java
Installation: Download Dialog_Grid_Demo.java to the plugins folder, or subfolder, and compile and run it using Plugins/Compile and Run.
Description: This plugin demonstrates how to use the addPanel() method of the GenericDialog class to add a grid of numeric fields to a dialog box.
