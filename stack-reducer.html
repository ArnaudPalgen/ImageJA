Stack Reducer

Author: Wayne Rasband (wsr at nih.gov)
History: 2009/06/02: First version
Source:Contained in the JAR file.
Installation: Download Stack_Reducer.jar to the plugins folder, run Help>Update Menus, and this plugin will be installed as the Image>Stacks>Reduce Size... command.
Description: This plugins reduces the number of images in a stack by a specified factor.
