Stack Reducer

Author: Wayne Rasband (wsr at nih.gov)
History: 2009/06/02: First version
Source: Contained in Stack_Reducer.jar.
Installation: This plugin is built into ImageJ 1.43t and later as the Image>Stacks>Tools>Reduce command.
Description: This plugins reduces the number of images in a stack by a specified factor.
