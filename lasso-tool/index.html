Lasso Selection Tool

Author: Johannes Schindelin (Johannes.Schindelin at gmx.de)
History: 2011/12/29: Modified to be compatible with ImageJ
Requires ImageJ 1.46d or later
Source: Java source is in lasso_tool.jar
Installation: Drag and drop lasso_tool.jar onto the "ImageJ" window, save it in the plugins/Tools directory, and run the Plugins>Tools>Lasso Tool command to add the Lasso Tool to the ImageJ toolbar. Double click on the Lasso Tool icon to display the configuration dialog box.
Description: This is a version of the Lasso Selection Tool (Plugins>Segmentation>Blow/Lasso Tool) in Fiji that has been modified to work with ImageJ.
