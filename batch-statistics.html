Batch Statistics

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/08/17: First version
2004/07/13: Works with stacks
Source: Batch_Statistics.java
Installation: Download Batch_Statistics.class to the plugins folder and restart ImageJ.
Description: This plugin displays statistics about all the images and stacks in a folder. The images can be in any format supported by ImageJ's Open command (Tiff, Jpeg, Gif, bmp, DICOM). Select any image in the file open dialog and the plugin will open and analyze all the images in the same folder. Use Analyze/Set Measurements to specify what gets measured. Use File/Save As/Measurements to save the results to a tab-delimited text file.
