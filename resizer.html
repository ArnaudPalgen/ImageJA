Resize Canvas

Author: Jeffrey Kuhn (jkuhn@ccwf.cc.utexas.edu)
The University of Texas at Austin
Date:2001/2/6
Source: Resize_Canvas.java
Installation: This plugin is built into ImageJ 1.31c or later as the Image/Adjust/Canvas Size command.
Description: This plugin changes the canvas size of an image or stack without scaling the actual image. The width and height may be either expanded or contracted. If the canvas size is increased, then the border is filled with the current background color. To change the background color, alt-click in the Image/Colors window. The position of the old image within the new canvas may also be specified.
