PlugIn Installer

Author: Dimiter Prodanov (dimiterpp at gmail.com)
History: 2009/07/28: First version
Source: PlugInInstaller_.java
Installation: The ability to install plugins by dragging from the file system or a browser window and dropping on the "ImageJ" window is built into ImageJ 1.43 or later.
Description: This plugin installs plugins packaged as jar files from the local file system or URLs. It supports specifically drag and drop from Mozilla Firefox browser tabs and can parse HTML for valid URLs.
