Benchmarks

Author: Wayne Rasband (wsr at nih.gov)
History: 2004/02/20: First version
2006/12/01: "Waves" displays frame rate in status bar; added results for 3GHz Mac Pro
Source:Contained in Benchmarks_.jar, which can be opened using a ZIP utility
Installation: Download Benchmarks_.jar to the plugins folder, or subfolder, restart ImageJ, and there should be a new Plugins/Benchmarks submenu containing nine commands.
Description: The plugins in this package run the simple image processing benchmarks that were in the Image/Benchmarks submenu in ImageJ 1.32a and earlier.
