PerkinElmer Reader

Author: Arne Seitz (arne.seitz at embl.de)
Advanced Light Microscopy Facility
EMBL Heidelberg
History: 2007/08/17: First version
Source: PerkinElmer_Reader.java
Installation: Download PerkinElmer_Reader.class to the plugins folder, restart ImageJ, and there will be a new "PerkinElmer Reader" command in the Plugins menu.
Description: This plugin opens raw files acquired with a PerkinElmer Ultraview (E)RS as a stack. The parameters (e.g. number of channels, number of slices, number of time points) of the image series are read from the cfg/tim file. It is a stripped-down version of the PerkinElmer Panel.
