Image_Rotator

Author: Dimiter Prodanov (dimiterpp at gmail.com)
History: 2012/05/28: First release
Source: Image_Rotator.java
Installation: Download Image_Rotator into the Image plugins folder, or subfolder, and compile an run it using the Plugins>Compile and Run command.
Description: This plugin rotates the active image around the center of mass of a drawn ROI. The image is rotated so that the X direction corresponds to the principal orientation axis of the ROI.
