Demontager

Author: Robert Clements (rclement at kent dot edu)
History: 2009/7/23: First version
Source: Demontager_.java
Installation: Download Demontager_.class to the plugins folder and restart ImageJ.
Description: This plugin converts a montaged image to an image stack based on the number of rows and columns input by the user. This is the opposite of what the Image/Stacks/Make Montage command does.
