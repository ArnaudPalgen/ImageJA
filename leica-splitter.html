Leica SP Splitter

Author: Joachim Walter (Walter at biz.uni-muenchen.de)
History: 2001/09/07: First version
2008/09/03: Updated to work with newer versions of ImageJ
Source: Leica_SP_Splitter.java
Installation: Download Leica_SP_Splitter.class and LeicaTiffDecoder.class to the plugins folder and restart ImageJ.
Description: Opens multichannel Leica SP files and saves the color channels to disk in separate Tiff stacks. The advantage, compared to the Leica_SP_Reader, is that one does not have to explicitly save the color channels if one simply splits them for use in some other software.
