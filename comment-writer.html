Comment Writer

Author: Francois Richard (richard@SCIENCE.UOTTAWA.CA)
Date:2000/9/29
Source: Comment_Writer.java
Installation: Copy Comment_Writer.class to the plugins folder and restart ImageJ.
Description: This is a simple plugin that allows you to add a comment to the ImageJ console. The comment defaults to the current date and time and image name (if one is open).
