LUT Lister

Author: Wayne Rasband (wayne@codon.nih.gov)
History: 2000/11/27: First version
2002/03/07: Writes to a separate "LUT" window
Source: Lut_Lister.java
Installation: Copy Lut_Lister.class to the plugins folder and restart ImageJ.
Description: Lists the contents of the current image's LUT (lookup table) to a text window. The contents of this window can be saved as a tab-delimited text file, edited in a spreadsheet, then reopened using the LUT_Importer plugin.
