Twain Scan

Author: Marcel Austenfeld (marcel.austenfeld at uni-bielefeld.de)
History: 2009/10/28: First version
Source: Scan_.java
Limitations: Does not work on Mac OS X.
Installation: Download ScannerPlugin.zip, extract the two jar files (Scan_.jar and uk.co.mmscomputing.device.twain.jar), and copy them to the plugins folder.
Description: This plugin uses the open source Twain library from www.mms-computing.co.uk to capture images from Twain devices such as scanners and USB cameras. It is bundled with the windows Twain library. The native library is automatically loaded from a jar file so only two jars have to be installed. The plugin has been successfully tested on Windows with a scanner and a digital camera. For Linux the Sane jar has to be installed (available on the same website) instead of the Twain library. The mms-computing library currently only works on Windows and Linux.
