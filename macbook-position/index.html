MacBook Position

Author: Jerome Mutterer (jerome.mutterer at ibmp-cnrs.unistra.fr)
History: 2010/02/26: Initial release
Limitation: Only runs on Apple MacBook Pros
Source: MacBook_Position.java
Installation: Download macbook-position.zip, extract the files, copy libUnimotionLib.jnilib to /Library/Java/Extensions, copy MacBook_Position.class and sms.jar to the ImageJ plugins folder, and restart ImageJ.
Description: This plugin is of very limited interest, but it's fun. It uses the JNI library at www.shiffman.net/p5/sms/ that lets you read your macbook's Sudden Motion Sensor. The jnilib file goes in /Library/Java/Extensions and sms.jar and the test plugin in the plugins folder. Run the plugin, hold your MacBook firmly, play!
