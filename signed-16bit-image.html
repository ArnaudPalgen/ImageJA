Signed 16-bit Image from Array

Author: Wayne Rasband (wsr at nih.gov)
History: 2006/02/16: First version
Source: Signed_16bit_Image.java
Installation: Download Signed_16bit_Image.java to the plugins folder and run it using Plugins/Compile and Run.
Description: This plugin demonstrates how to create a signed 16-bit image from an array. The signed image is created by adding 32768 and then creating a calibration function that subtracts 32768.
