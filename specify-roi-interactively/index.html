Specify ROI Interactively

Author: Jeffrey Kuhn (jkuhn at ccwf.cc.utexas.edu)
Anthony Padua (padua001 at mc.duke.edu)
Wayne Rasband (wsr at nih.gov)
History: 2013/04/24: First release
Source: Specify_ROI_Interactively.java
Installation: Download Specify_ROI_Interactively.java into the ImageJ/plugins folder and compile and run it using the Plugins>Compile and Run command.
Description: This plugin demonstrates how to interactively set the position and size of rectangular and oval selections using a dialog with sliders. It is based on the internal plugin that implements the Edit>Selection>Specify ROI command.
