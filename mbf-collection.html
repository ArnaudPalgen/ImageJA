MBF "ImageJ for Microscopy" Collection

Author: Tony Collins
History: 2007/12/10: Updated to October 2007 release
2009/09/21: Updated to October 2008 release
Source: The Java source code for most of the plugins is in the ZIP archive in the plugins/source directory.
Installation: Download mbf-plugins.zip (15 MB), extract the contents, and copy the resulting "plugins" folder into the ImageJ folder. Before doing this, you may want to rename the existing "plugins" folder.
Description: This is the "ImageJ for Microscopy" collection of ~200 plugins collated and organized by Tony Collins at the McMaster Biophotonics Facility. An online manual describing the collection is availble at www.macbiophotonics.ca/imagej/.
