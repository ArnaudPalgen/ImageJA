Image Layering Toolbox

Author: Karl Schmidt (kfschmidt@bwh.harvard.edu)
Date: 2002/06/19
Requires: Java 2
Source: Included in the ZIP archive
Installation: Download IMAGE_LAYERING_PLUGIN.zip and unzip the files to a temporary folder. Once unzipped, copy the contents of the plugins folder to the plugins folder in the ImageJ folder.
Description: This plugin does transparent image composition and creates contour maps. Documentation, in PDF format, is in the ZIP archive.
