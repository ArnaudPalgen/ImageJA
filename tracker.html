Object Tracker

Author: Wayne Rasband (wsr at nih.gov)
Date:2000/6/21
Requirements:ImageJ 1.17y or later
Limitations:Only tracks two objects. Requires a thresholded 8-bit stack.
Source: Tracker_.java
Installation: Copy Tracker_.class to the plugins folder and restart ImageJ.
Description: This plugin uses ImageJ's particle analyzer to track one or two objects through a stack. For each stack frame, it displays the number of objects, the coordinates of the first two objects, and the distance between the first two objects. It also plots distance vs. frame. It can be customized to track more than two objects. A test stack, contributed by Ross Lang, is provided.
See Also: MultiTracker, Manual Tracking, SpotTracker, MTrack2, ParticleTracker, MTrackJ and Measure Track plugins
