RGB to CMYK

Authors: Stephan Saalfeld (saalfeld at MPI-CBG.DE)
Wayne Rasband (wsr at nih.gov)
History: 2010/05/15: Initial release
Source: RGB_to_CMYK.java
Installation: Drag and drop RGB_to_CMYK.class onto the "ImageJ" window.
Description: This is a plugin version of Stephan Saalfeld's BeanShell script at
   http://pacific.mpi-cbg.de/wiki/index.php/RGB_to_CMYK
which converts an uncalibrated linear RGB image into an uncalibrated linear 32-bit CMYK stack.
