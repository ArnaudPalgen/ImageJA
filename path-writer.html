Path Writer

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/11/04: First version
2002/09/10: Bug fixes
2004/03/27: Uses four decimal places; origin at lower left only if "Invert Y Coordinates" checked in Analyze/Set Measurements.
Source: Path_Writer.java
Installation: Download Path_Writer.class to the plugins folder and restart ImageJ.
Description: This plugin saves a list of evenly spaced (one pixel) x-y coordinates along the current line or area selection to a tab-delimited text file. The origin of the coordinate system is the lower left hand corner if "Invert Y Coordinates" checked in Analyze/Set Measurements.
See Also: ROI Importer plugin
