HyperVolume Opener

Author: Joachim Walter (walter at biz.uni-muenchen.de)
History: 2003/11/24: First version
2003/12/20: Adds "Scale:" option; macro recording works
2005/06/17: Improved and built into Image5D
Source: Hypervolume_Opener.java
Installation: The HyperVolume Opener is included with Image5D.
Description: This is an extended version of the internal FolderOpener plugin (File/Import/Image Sequence) that imports all the slices of any stacks it encounters. It allows the user to import a series of 3D stacks and view them with Image5D.
