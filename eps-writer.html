EPS Writer

Author: Johannes Schindelin (Johannes.Schindelin at GMX.DE)
History: 2006/05/02: First version
Limitation: Does not include a preview image in EPS file
Source: EPS_Writer.java
Installation: Download EPS_Writer.class to the plugins folder, or subfolder, restart ImageJ, and there will be a new "EPS Writer" command in the Plugins menu, or submenu.
Description: This plugin saves the current image in EPS (Encapsulated PostScript) format.
See Also: Export EPS plugin
