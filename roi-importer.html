ROI Importer

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/11/15: first version
2003/06/27: Uses "handles" based on vertex spacing
2005/02/04: Works with ROIs with negative coordinates
Source: ROI_Importer.java
Installation: Download ROI_Importer.class to the plugins folder and restart ImageJ.
Description: This plugin imports a two column text file, such as those created by File/Save As/XY Coordinates, as a polygon selection. The selection is displayed in the current image or, if the current image is too small, in a new blank image.
