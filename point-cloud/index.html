Export Point Cloud

Author: Jonathan B. Glass (jonathan.b.glass at gmail.com)
History: 2010/04/11: Initial release
Source: In Point_Cloud.jar
Installation: Drag and drop Point_Cloud.jar onto the "ImageJ" window (drag and drop installation requires ImageJ 1.43).
Description: This plugin turns a stack of CT scans into a point cloud viewable in Excel, Matlab or Paraview. There is a PDF file (point_cloud.pdf) with more information.
