Gif Stack Writer

Author: Ryan Raz (http://rraz.ca)
History: 2002/03/07: First version
2007/06/28: Bug fixes; dialogs removed; packaged as JAR
Source:Contained in Animated_Gif.jar
Installation: This plugin is built into ImageJ 1.39d and later as the File>Save As>Gif command.
Description: This plugin saves a stack as an animated gif. The JAR file also contains the Animated_Gif_Reader plugin, which is installed as the File/Import/Animated Gif ... command.

The original version of this plugin (Gif_Stack_Writer.java), which has a dialog box with many options, is still available. It will need to compiled using Plugins>Compile and Run.

See Also: Animated Gif Reader plugin
