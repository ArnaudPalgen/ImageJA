Source: Anonymize_IJ_DICOM.java
Installation: Download Anonymize_IJ_DICOM.class to the plugins folder and restart ImageJ or click Help > Update Menus.
Description:

This plugin is designed to anonymize the DICOM metadata of an image or stack after it has been opened in ImageJ. This information can be visualised by going to Image > Info but includes patient specific information. The plugin allows the user to select which tags should be anonymized, and to choose the text with which the patient information is replaced.


See also: DICOM Rewriter (anonymizes DICOM files in a folder before import into ImageJ).
