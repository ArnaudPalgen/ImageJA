RGB Stack Splitter

Author: Wayne Rasband (wayne@codon.nih.gov)
History: 2000/04/13: first version
2001/05/30: works with single images
2001/12/12: faster
2002/12/06: built into ImageJ 1.29t
Source: RGB_Stack_Splitter.java
Installation: This plugin is built into ImageJ 1.29 and later as the Image/Color/RGB Split command.
Description: Splits an RGB image or stack into three 8-bit grayscale images or stacks. The RGB_Stack_Merge plugin does the reverse operation.
