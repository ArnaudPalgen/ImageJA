Blob Labeler

Author: Dimiter Prodanov (dimiterpp at gmail.com)
History: 2013/03/22: First release
Source: Blob_Labeler.java
Installation: Drag and drop Blob_Labeler.class onto the "ImageJ" window.
Description: The plugin labels connected blobs of pixels, which are different from the background color. The labeling functions support both 4- and 8- connectivity.

Optionally the outlines of the objects can be added to the RoiManager.

Another option is edge correction where a 1-pixel thick border is added to x=0 and y=0 axes.

The code for 4-connected labeling is based on homepages.inf.ed.ac.uk/rbf/HIPR2/labeldemo.htm
