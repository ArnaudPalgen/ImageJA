Stack Builder

Author: Nicolas Roggli, University of Geneva, Switzerland
and Wayne Rasband
History: 2001/08/31: First version
2002/03/26: Stack set to size of largest image
2002/06/24: 16-bit images converted to a 16-bit stack
Source: Stack_Builder.java
Installation: Download Stack_Builder.class to the plugins folder and restart ImageJ.
Description: This plugin creates a stack from all open images, which may differ in size and type. For images with selections, only the selection is used. The size of the stack is the same as the largest image or selection. Smaller images are padded with the current foreground color.

In ImageJ 1.42 and later, the built in Image>Stacks>Convert Images to Stack command can also create a stack from images that differ in size or type.
