Stack Maker

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/1/30: First version
2005/07/04: Macro recordable
Source: Stack_Maker.java
Installation: This plugin is built into ImageJ 1.43t and later as the Image>Stacks>Tools>Montage to Stack command.
Description: This plugin creates a w*h slice stack from an wxh image montage. This is the opposite of ImageJ's Image/Stacks/Make Montage command.
