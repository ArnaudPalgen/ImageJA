MRI Analysis Calculator

Author: Karl Schmidt (kfschmidt@bwh.harvard.edu)
Date: 2002/06/19
Source: Included in the ZIP archive
Installation: Download MRI_CALC_PLUGIN.zip and unzip the files to a temporary folder. Once unzipped, copy the contents of the plugins folder to the plugins folder in the ImageJ folder.
Description: This plugin performs pixelwise T1, T2, perfusion and (not fully implemented) diffusion calculations on MR Image data. Documentation, in PDF format, is in the ZIP archive.
