PGM/PPM/PNM Writer

Author: Johannes Schindelin (Johannes.Schindelin at GMX.DE)
History: 2006/03/09: First version
Source: PNM_Writer.java
Installation: This plugin is built into ImageJ 1.36a and later as the File/Save/PGM... command.
Description: This plugin saves 8-bit images in PGM (portable graymap) format and RGB images in PPM (portable pixmap) format. These formats, along with PBM (portable bitmap), are collectively known as the PNM format. More information about these formats can be found in the Wikipedia.

Note that ImageJ 1.35 may throw an exception when reading PGM, PPM or PBM files. This is fixed in ImageJ 1.36a.
