Concatenate

Author: Jonathan Jackson (j.jackson at ucl.ac.uk)
History: 2006/06/25: First version
2007/04/23: Supports creating an 'Image5D' with color channels
Source: Concatenate_.java
Installation: Download Concatenate_.class to the plugins folder and restart ImageJ.
Description: This plugin concatenates two or more images or stacks that have the same width, height and data type.  If the Image5D plugin is installed, there is the option to create an 'Image5D' from the concatenated stack.
