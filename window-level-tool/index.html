Window Level Tool

Author: James Norman (james.norman at gmail.com)
History: 2011/12/29: Updated to a PlugInTool
Requires ImageJ 1.46d or later
Source: Window_Level_Tool.java
Installation: Drag and drop Window_Level_Tool.class onto the "ImageJ" window.
Description: This is an upgraded version of the Window Level Tool at
alger.bmap.ucla.edu/WindowLevelMacro
To use it, select the "WL" tool and click and drag on an image to adjust its window and level.
