Color Profiler

Author: Dimiter Prodanov (D.Prodanov at lumc.nl)
History: 2004/10/24: First version
2005/06/28: Plot and statistics are updated if the ROI is moved/resized
Source:Contained in Color_Profiler.jar, which can be opened using a ZIP utility
Installation: Download Color_Profiler.jar to the plugins folder, or subfolder, restart ImageJ, and there will be a new "Color Profiler" command in the Analyze/Tools menu.
Description: This plugin provides the same functionality as the Analyze/Plot Profile command but for RGB images. It provides superior functionality to the RGBProfiler.
