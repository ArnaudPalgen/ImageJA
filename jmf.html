JMF Movie Reader

Author: Wayne Rasband (wsr at nih.gov)
Date: 2001/05/21
Limitations: Only opens a few of the many QuickTime and AVI video formats. Opens RGB stacks saved using File>Save As AVI but not 8-bit stacks. Requires the 1.8MB jmf.jar file.
Source: JMF_Movie_Reader.java
Installation: Download JMF_Movie_Reader.class to the plugins folder and jmf.jar (1.8MB) to the Java extensions folder. On Windows and Linux this folder is at ImageJ/jre/lib/ext, on Mac OS X it is at /Library/Java/Extensions, and on Mac OS 9 it is at /System Folder/Extensions/MRJ Libraries/MRJClasses. The jmf.jar file is also available as part of the Java cross-platform ZIP file at the JMF Web site.
Description: This plugin use the JMF (Java Media Framework) to open QuickTime and AVI movies in Cinepak, JPEG or uncompressed RGB format.
See Also: AVI Reader, QuickTime Movie Opener and JMF Player.
