Overlay Brush Tool

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/02/19: First release
Requires ImageJ 1.46h or later
Source: Overlay_Brush_Tool.java
Installation: Drag and drop overlay_brush_tool.jar onto the "ImageJ" window, save it in plugins/Tools, then select Overlay Brush Tool from the toolbar's ">>" drop down menu.
Description: This tool is a brush that draws on a non-destructive image overlay. Double click on the tool icon to open the options dialog box.
