Overlay Brush Tool

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/02/16: First release
Requires ImageJ 1.46g or later
Source: Overlay_Brush_Tool.java
Installation: Drag and drop Overlay_Brush_Tool.class onto the "ImageJ" window, save it in plugins/Tools, then select Overlay Brush Tool from the toolbar's ">>" drop down menu.
Description: This tool is a brush that draws on a non-destructive image overlay. Use the color picker to to specify the drawing color and double click on the tool icon to set the brush width.
