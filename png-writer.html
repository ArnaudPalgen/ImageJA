PNG Writer

Author: Wayne Rasband (wsr at nih.gov)
History: 2004/02/26: First version
Requires: Java 1.4 or later and ImageJ 1.31 or later
Source:Contained in PNG_Writer.jar, which can be opened using a ZIP utility
Installation: This plugin is built into ImageJ 1.37b and later as the File/Save As/PNG...command.
Description: This plugin saves in PNG format using the ImageIO classes available in Java 1.4 or later. Use the File/Open command to open PNG files.
