Import Bruker NMR Files

Author: Bertram Manz (bertram.manz at ibmt.fraunhofer.de)
Fraunhofer-Institut fur Biomedizinische Technik
St. Ingbert, Germany
History: 2004/07/29: First version
2004/08/04: Revised version
2008/04/22: Bug fixes
Source:Contained in BrukerOpener.jar, which can be opened using a ZIP utility
Installation: Download BrukerOpener.jar to the plugins folder, or subfolder, restart ImageJ, then open Bruker files using File/Open or drag and drop.
Description: This plugin reads raw and processed data files from a Bruker NMR spectrometer. The information about file size and data format is taken from the header files in the same directory. Open fid, ser, 2dseq, 2rr, 2ii, 3rrr, 3iii files using the File/Open... command or drag and drop.

This plugin requires a version of the HandleExtraFileTypes plugin that supports Bruker files.
