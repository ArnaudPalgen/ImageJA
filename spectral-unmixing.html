Spectral Unmixing Plugins

Author: Joachim Walter (walter at biz.uni-muenchen.de)
History: 2004/11/03: First version
2006/04/19 (v1.3): Measure Mixing Matrix works with SyncWindows v1.5
Requires:ImageJ 1.37b or later
Source:Contained in Spectral_Unmixing.jar, which can be opened using a ZIP utility
Installation: Have the plugin Sync_Windows_1.3 or higher installed. Put Spectral_Unmixing.jar and Jama-1.0.1.jar into the ImageJ plugins-folder or a subfolder and restart ImageJ. A submenu "SpectralUnmixing" of the Plugins menu appears, which contains the entries "Display Mixing Matrix", "Measure Mixing Matrix" and "Unmix".
Description: Spectral unmixing is an algorithm, which is applied to fluorescence images to correct for spectral bleedthrough between different color channels. E.g. the emission spectra of the fluorochromes CFP and YFP partially overlap, and CFP signals also appear in the YFP fluorescence channel. This overlap can be corrected by spectral unmixing.

Additional documentation in PDF format is available.

The public domain JAMA library, a basic linear algebra package for Java, is used for matrix inversion.
