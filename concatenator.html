Concatenator

Author: Wayne Rasband (wsr at nih.gov)
History: 2000/11/01: First version
2002/02/19: Renamed "Concatenator"; added dialog 2009/10/12: Works in batch mode macros
Source: Concatenator_.java
Installation: This plugin is built into ImageJ 1.43t and later as the Image>Stacks>Tools>Concatenate command.
Description: This plugin concatenates two images or stacks that have the same width, height and data type.
