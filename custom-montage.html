Custom Montage Plugin

Author: Alex Watson (david dot watson at nih dot gov)
History: 2011/06/23: First version
Requires:ImageJ 1.44o or later, which is the oldest version tested.
Source:Contained in Custom_Montage.jar, which can be opened using a ZIP utility
Installation: Download Custom_Montage.jar to the plugins folder, or subfolder, restart ImageJ, and there should be two new plug-ins under the 'Graphics' submenu.
Description: This plugin's purpose is to create an aesthetically-pleasing montage image for publications from LSM files and many other types of image stacks.
