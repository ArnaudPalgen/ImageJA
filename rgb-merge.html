RGB Stack Merge

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/02/12: first version
2001/06/30: works with stacks with inverted LUTs
2001/12/12: faster
2002/06/18: works with RGB input stacks
2002/12/06: built into ImageJ 1.29t
2007/03/20: source code updated
Source: RGB_Stack_Merge.java
Installation: This plugin is built into ImageJ 1.29 and later as the Image/Color/RGB Merge command.
Description: Combines one, two or three 8-bit grayscale or RGB images or stacks into one RGB image or stack.
