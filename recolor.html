RGB Recolor

Author: Kieran Holland (holki659@student.otago.ac.nz)
Date:2001/2/12
Source: RGB_Recolor.java
Installation: Download RGB_Recolor.class to the plugins folder and restart ImageJ.
Description: This plugin allows linear alteration of the colors in the red, green and blue planes of an RGB image or stack, where:
New color = Old Color*Scaling Factor + Constant
If the new color is greater than 255 it is set to 255. If it is less than 0 it is set to 0.
