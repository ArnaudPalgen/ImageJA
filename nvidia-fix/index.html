nVidia Fix

Author: Dimiter Prodanov (dimiterpp at gmail.com)
History: 2011/03/28: First release
Limitations: Windows only
Source: nVidia_Fix.java
Installation: Drag and drop nVidia_Fix.class onto the "ImageJ" window.
Description: Run this plugin to add the "-Dsun.java2d.noddraw=true" command line option to the ImageJ Windows configuration file (ImageJ.cfg). This may fix display problems caused by a buggy nVidia display driver. You will need to restart ImageJ after running this plugin. Delete the "ImageJ.cfg" if there is a problem.
