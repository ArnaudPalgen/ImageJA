Example Plot

Authors: Hajime Hirase and Wayne Rasband
Source: Example_Plot.java
Installation: Download Example_Plot.class to the plugins folder and restart ImageJ.
Description: This plugin provides an example of how to use ImageJ's Plot class. Plots can also be generated using macros. Refer to the ExamplePlots macro file for examples.
See Also: Calc CCD SNR plugin
History: 2001/05/30: First version
2002/04/05: Added color
2003/01/15: Added label, wider lines, bigger fonts
2007/01/13: Uses Plot class instead of PlotWindow
2010/01/13: Always draws grid lines regardless of Edit>Options>Profile Plot "Draw grid lines" setting
