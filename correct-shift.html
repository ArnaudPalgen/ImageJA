Correct X Shift

Author: Yannick KREMPP (yannick.krempp at unil.ch)
Cellular Imaging Facility
Lausanne, Switzerland
History: 2006/03/02: First version
Source:Contained in the JAR file. To open a JAR file, change the extension from ".jar" to ".zip" and double click on it.
Installation: Download correctXshift_.jar to the plugins folder, or subfolder, restart ImageJ, and there will be a new "Correct X Shift" command in the Plugins menu, or submenu.
Description: This plugin is intended to correct the shift that occurs between lines when working with bidirectional acquisition with the LSM 510 confocal microscope. It can work on any 8-bit image or stack. The advantage is to be able to do this tuning offline instead of doing it during acquisition.
See Also: X Shifter plugin
