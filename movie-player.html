Source: QT_Movie_Player.java
Installation: Download QuickTime_Plugins.jar to the plugins folder, or subfolder, restart ImageJ and there will be a new File/Import/QuickTime Player command. This JAR file also installs three other QuickTime plugins:
  • QuickTime Capture (File/Import/Video)
  • QT Movie Opener (File/Import/Using QuickTime)
  • QT Movie Writer (File/Save As/QuickTime Movie)
Note that this JAR file is included with the Mac OS X version of ImageJ, version 1.34 and later.
Description: Plays a QuickTime movie in a window. Requires QuickTime and QuickTime for Java, which are preinstalled on Macs. Windows users must install QuickTime 7 and copy QTJava.zip to the Java extensions folder (ImageJ\jre\lib\ext).

This plugin uses a standard QuickTime controller bar for starting and stopping the movie, moving throught it, and adjusting the volume. In addition, you can start and stop the movie by pressing the space bar, move foreward or backward by a single frame using the arrow keys, and capture a frame by pressing "g" (grab). Also note that clicking on a running movie stops it and double clicking starts it.
