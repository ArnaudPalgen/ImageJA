Animated Gif Reader

Authors: Kevin Weiner, FM Software
Wayne Rasband
History: 2001/06/05: First version
2007/06/28: Packaged as a JAR file
Source: Animated_Gif_Reader.java
Installation: Download Animated_Gif.jar to the plugins folder, or subfolder, restart ImageJ, and there will be new File/Import/Animated Gif... and File/Save As/Animated Gif... commands.
Description: This plugin opens an animated Gif file as an RGB stack. Also opens single Gif images. The JAR file also contains the Animated_Gif_Writer plugin, which is installed as the File/Save As/Animated Gif ... command.
See Also: Gif Stack Writer plugin
