Color Chooser

Author: Daniel Marsh (marshd@ninds.nih.gov)
Date: 2002/07/15
Requires: Java 2
Limitations: On Mac OS X, you have to click twice on a tab to switch to a different panel.
Source: Color_Chooser.java
Installation: Download Color_Chooser.class to the plugins folder and restart ImageJ.
Description: This plugin uses the Swing JColorChooser class to provide a palette of colors to choose from. It can be used to set either the foreground or background color.
