Arrow Tool

Authors: Jean-Yves Tinevez (jean-yves.tinevez at pasteur.fr)
Johannes Schindelin (Johannes.Schindelin at gmx.de)
History: 2011/12/30: Modified to work with ImageJ
Requires ImageJ 1.46d or later
Source: Java source is in arrow_tool.jar
Installation: Drag and drop arrow_tool.jar onto the "ImageJ" window, save it in the plugins/Tools directory, then run the Plugins>Tools>Arrow Tool command to add the Arrow Tool to the ImageJ toolbar. Double click on the Arrow Tool icon to display the configuration dialog box.
Description: This is a version of the Arrow Tool in Fiji (Image>Annotate>Arrow), modified to work with ImageJ.

See also: The arrow tool built into ImageJ, accessed by right-clicking on the line tool.
