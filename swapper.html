Byte Swapper

Author: Wayne Rasband (wsr at nih.gov)
History: 2001/10/25: First version
2005/06/22: Fixed bug that caused byte swapping of some 32-bit pixels to fail
2006/01/03: Works with signed 16-bit images
Source: Byte_Swapper.java
Installation: Download Byte_Swapper.class to the plugins folder and restart ImageJ.
Description: This plugin reverses the byte order of 16 and 32 bit images. To export an image or stack in little-endian byte order, check "Export Raw in Intel Byte Order" in Edit/Options/Input/Output and use the File/Save As/Raw Data command (requires 1.37e or later).
