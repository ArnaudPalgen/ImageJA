Results_Importer

Author: Dimiter Prodanov (dimiterpp at gmail.com)
History: 2011/12/27: First release
Source: Results_Importer.java
Installation: Drag and drop Results_Importer.class onto the "ImageJ" window.
Description: This plugin imports numeric values from a text window into the ResultsTable. This is useful, for example, for importing histogram or line profile data.
