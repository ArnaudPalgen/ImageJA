Installation:

Download nd_stack_builder.class to the plugins folder and restart ImageJ.

Description:

This plug-in reads *.nd files generated by the option "Multidimensional acquisition" of the Metamorph software.

The plug-in may be used to build temporal stacks for each parameter of a 2/3D+ n colours + n positions + time dataset (one temporal stack per colour for each position). In case the original images are stacks, the user is prompted to select the way the projection should be done (choice of projection method and start/end slice). Finally, if the pointed file is not a *.nd file, the plug-in will generate a stack with all images (or projections of the stacks) contained in the current folder: it may be used as a "batch projector" plug-in. In both cases, resulting stack(s) will have its slices labelled with the original filename (required for 3D functions of the new version of Manual Tracking).
