Apply Formula

Author: Wayne Rasband (wsr at nih.gov)
History: 2012/08/29: First release
Source: Apply_Formula.java
Installation: Download Apply_Formula.java into the ImageJ/plugins folder and compile and run it using the Plugins>Compile and Run command.
Description: This plugin demonstrates how to apply a formula to every pixel in an image or stack. It works with all image types (except RGB), it is multi-threaded when processing stacks, and it supports Undo.
