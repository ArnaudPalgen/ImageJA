Thread Killer

Author: Johannes Schindelin (Johannes.Schindelin at GMX.DE)
History: 2008/07/16: First version
Source: Thread_Killer.java
Installation: Download Thread_Killer.class to the plugins folder, or subfolder, restart ImageJ, and there will be a new "Thread Killer" command in the Plugins menu, or submenu.
Description: Use this plugin to abort a plugin that does not quit when the escape key is pressed.
See Also: Escape plugin - Demonstrates how a plugin can abort itself when the the escape key is pressed
