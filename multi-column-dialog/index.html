MultiColumnDialog

Author: Wayne Rasband (wsr at nih.gov)
History: 2010/052/11: Initial release
Requires: ImageJ 1.44a or later
Source: MultiColumnDialog.java
Installation: Download MultiColumnDialog.java to the plugins folder and compile and run it using the Plugins>Compile and Run command.
Description: This plugin demonstrates how to display checkbox groups in GenericDialogs. It requires ImageJ 1.44a or later. The dialog box it displays is modeled after the LOCI Bio-Formats Import Options dialog box.
