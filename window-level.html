Window/Level Adjuster

Author: Walter O'Dell PhD, wodell@rochester.edu
History: 2002/11/13: First version
Source: The Java source code is included in the window-level.zip archive.
Installation: Download window-level.zip and extract the contents to the ImageJ folder. Requires a ZIP utility such as WinZIP (PC) or Stuffit Expander (Mac).
Description: Adjusts window and level of the active image in the manner similar to that implemented on medical image terminals -- replacement for brightness/contrast adjustment.

Works on Stacks color images also. Because it uses the GenericRecallableDialog class, keyboard arrow keys act on the most-recently mouse-touched scrollbar.

See Also: Image>Adjust>Window/Level command
