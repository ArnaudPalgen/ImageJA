Radial Grid

Author: Adam Baker (tabaker at u.arizona.edu)
History: 2003/11/26: First version
Source: Radial_Grid.java
Installation: Download Radial_Grid.class to the plugins folder and restart ImageJ.
Description: This plugin draws a radial grid over an image. The user specifies the origin, length, and color of the radii, as well as the intervals at which they appear. The defaults are set to work for a certain ultrasound machine, and are not anticipated to be generally useful (of course, they can be changed in the source code).
