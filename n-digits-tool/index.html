N Digits Tool

Author: Johannes Schindelin (Johannes.Schindelin at GMX.DE)
History: 2012/08/09: Initial release
Requires ImageJ 1.46d or later
Source: N_Digits_Tool.java
Installation: Drag and drop N_Digits_Tool.class onto the "ImageJ" window, save it in the plugins/Tools directory, and run the Plugins>Tools>N Digits Tool command to add the N Digits Tool to the ImageJ toolbar.
Description: This tool displays, in the status bar, the location and pixel value as the cursor is moved over an image, overriding the built in ImageJ function that does this. Double click on the tool icon to set the number of decimal places (default is 3).
